#Quote Generator

Проект представляет собой консольное приложение для генерации случайных цитат с возможностью фильтрации по категориям.
Цель проекта — смоделировать процесс командной разработки с использованием Agile и Scrum.
В рамках лабораторной работы используется GitHub Issues для формирования Product Backlog и управления спринтом.
Работа разбита на отдельные инкременты, отражающие функциональные и нефункциональные требования.
